Dev Tools Sort Items Interactive Activity

The CidiLabs DesignPLUS Order Items tool (often referred to as Sort Items or Sorting Interactions) is an interactive, drag-and-drop activity designed for the Canvas Learning Management System. It allows instructors to create engaging, formative learning experiences that require students to arrange items in a specific, correct order.

  Features

Key Features

  • Interactive Learning: Students physically drag and drop items to organize them, which encourages higher-order thinking and improves long-term retention of information.
  • Immediate Feedback: The tool provides instant feedback to students, allowing them to verify their understanding immediately.
  • Accessibility: As a DesignPLUS feature, these sorting interactions are designed to be accessible, ensuring they can be used by all students.
  • Easy Implementation: The tool is part of the DesignPLUS Sidebar (accessed via the "rocket ship" icon in the Canvas Rich Content Editor), allowing for quick creation without needing to write custom HTML or JavaScript.
  • Use Cases: It is ideal for sequencing historical events, organizing steps in a process, sorting data, or categorizing concepts.
