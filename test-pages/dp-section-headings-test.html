Current CSS Classes Involved:

  • .dp-has-icon - Main container for heading + icon layout
  • h3.dp-has-icon - Specific h3 styling
  • .fas, .fa - FontAwesome icon classes
  • .dp-icon-content - Icon content wrapper (usually hidden)
  • border-color - TSTC brand color integration

Areas to Address:

  • Icon and text alignment
  • Consistent spacing between icon and text
  • Border treatments and color consistency
  • Responsive behavior on small screens
  • Typography hierarchy with surrounding content
  • FontAwesome version compatibility

Canvas Pattern Analysis:

  • Common structure: <h3 class="dp-has-icon" style="border-color: #1d2757;">
  • Icon pattern: <i class="fas fa-[icon]" style="border-color: #1d2757;" aria-hidden="true">
  • Hidden span: <span class="dp-icon-content" style="display: none;">&nbsp;</span>
  • TSTC brand color (#1d2757) consistently applied
