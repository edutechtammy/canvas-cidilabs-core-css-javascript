dp-image System Requirements:

  • ✓ .dp-image-rounded-5 / .dp-image-rounded-10 - Border radius variants
  • ✓ .dp-image-bordered - Border styling
  • ✓ .dp-image-shadow - Box shadow effect
  • ✓ .dp-image-padded - Padding around image
  • ✓ .dp-full-width - Full width responsiveness
  • ✓ .dp-ratio-1-1 / .dp-ratio-4-3 - Aspect ratio control
  • ✓ .dp-popup-image - Click to enlarge functionality
  • ✓ .object-fit-cover - CSS object-fit integration
  • ✓ .dp-order-figure - Ordered list styling for image sequences

Goal: Build Canvas-accurate image styling with responsive behavior and optional popup functionality.
