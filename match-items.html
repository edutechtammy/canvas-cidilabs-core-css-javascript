Dev Tools Match Items Interactive Activity

CidiLabs Match Items: Drag and drop matching activities designed to enhance engagement through interactive pairing exercises within the Canvas Learning Management System.

  Features

Key Features and Benefits

  • Interactivity: Improves learning through active participation in matching related concepts.
  • Accessibility: Fully accessible to all users, including keyboard navigation support.
  • Customization: Supports text content and can be configured for various matching activities like vocabulary pairs, concept relationships, or classification exercises.
  • DesignPLUS Integration: Part of the Design Tools for Canvas, allowing for easy creation without needing to know HTML.
  • Visual Feedback: Provides immediate feedback when items are successfully matched with visual indicators.

Usage in Canvas

Functionality: Students click on items in the "Items to Match" area. When two items form a correct pair, they receive visual confirmation and are moved to the "Matched Items" section.

Scoring: Tracks completion progress and can display final scores with reset functionality.

Placement: Can be added to pages, assignments, or discussions within Canvas.

The base match items template has been adapted with:

  • TSTC branding colors (Texas Blue background with red borders)
  • Enhanced visual hierarchy with proper icon integration
  • Responsive design for various screen sizes
